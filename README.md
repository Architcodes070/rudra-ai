# 🔥 Rudra AI

> Not just AI. Controlled Intelligence.

Rudra AI is a powerful, intelligent chatbot built using Streamlit and OpenRouter APIs.  
It provides a clean Chatbot-like interface with smart responses, chat history, and file analysis capabilities.

---

## 🚀 Features

- 💬 Real-time AI Chat
- 🧠 Smart Responses (Auto Model Selection)
- 🗂 Multiple Chat Sessions
- ⚡ Fast & Lightweight UI
- 🔐 Secure API Key Handling
- 🎯 Custom AI Personality (Rudra AI Mode)

---

## 🛠 Tech Stack

- Python 🐍
- Streamlit ⚡
- OpenRouter API 🤖
- dotenv (Environment Variables)
